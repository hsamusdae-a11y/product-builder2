## 🚀 사용 방법

### 1️⃣ 로컬 실행
```bash
# 압축 해제
unzip bible-app-integrated.zip
cd bible-app-integrated

# 로컬 서버 실행 (Python)
python -m http.server 8000

# 또는 (Node.js)
npx serve .

# 브라우저에서 http://localhost:8000 접속
```

### 2️⃣ Firebase 배포 (무료 호스팅)
```bash
# Firebase CLI 설치
npm install -g firebase-tools

# 로그인
firebase login

# 프로젝트 초기화
firebase init hosting

# 배포
firebase deploy
```

**상세 가이드**: `README-FIREBASE.md` 참고

## 🔐 인증 흐름

1. **첫 방문** → 로그인/회원가입 화면
2. **회원가입** → 필수 정보 입력 (이메일, 비밀번호, 성명 등)
3. **로그인** → 세션 저장 (localStorage)
4. **메인 앱 접근** → 모든 기능 사용 가능
5. **권한 확인** → 게시판 글쓰기 (추천 5개 이상 OR 레벨 2+)

## 📝 설교 작성 흐름

### 기본 흐름 (5단계)
1. **주제/성구 입력** → "사랑" 또는 "요한복음 3:16"
2. **제목 추천** → AI가 3가지 제목 생성 → 선택
3. **시간 선택** → 5분/10분/30분
4. **3대지 자동 생성** → 서론, 본론 3, 결론
5. **찬송가 추천** → 관련 찬송가 5곡

### 최종 편집
- 제목, 본문 말씀, 개요, 내용 직접 수정 가능
- **저장** → 내 설교에 저장
- **인쇄** → 프린터 출력
- **다운로드** → TXT/JSON 로컬 저장
- **초기화** → 다시 작성

### 내 설교 관리
- **보기** → 팝업으로 내용 확인
- **수정** → 편집 폼으로 불러오기
- **삭제** → 영구 삭제
- **인쇄** → 프린터 출력

## 💾 데이터 저장 (LocalStorage)

### 저장되는 데이터
- `users` - 회원 정보
- `currentUser` - 현재 로그인한 사용자 이메일
- `sermons` - 개인 설교 목록
- `boardPosts` - 게시판 글 목록

### 데이터 백업
```javascript
// 브라우저 콘솔에서 실행
// 백업
const backup = {
    users: localStorage.getItem('users'),
    sermons: localStorage.getItem('sermons'),
    boardPosts: localStorage.getItem('boardPosts')
};
console.log(JSON.stringify(backup));

// 복원
localStorage.setItem('users', backup.users);
localStorage.setItem('sermons', backup.sermons);
localStorage.setItem('boardPosts', backup.boardPosts);
```
