# AI 성경 노트 애플리케이션 명세서 (통합 버전)

## 1. 개요
이 애플리케이션은 목회자와 성경 연구자들을 위한 통합 플랫폼으로, 성경 연구, 설교 작성, 찬양 검색 및 성도 간의 소통(게시판) 기능을 제공합니다.

## 2. 디자인 및 스타일 가이드
- **UI**: 깨끗하고 세련된 카드 레이아웃 기반의 화이트/블루 테마
- **반응형**: 모바일과 데스크톱 모두에서 최적화된 화면 제공
- **컴포넌트**: 그림자 효과가 적용된 카드, 애니메이션 버튼, 직관적인 아이콘 사용

## 3. 기능 상세

### 1. 홈 (Home)
- 앱의 주요 기능을 한눈에 볼 수 있는 퀵 링크 카드 제공

### 2. 오늘의 말씀 (Daily Verse)
- 실제 성경 데이터(`public/bible/`)에서 무작위로 구절을 선정하여 표시

### 3. 성구 검색 (Bible Search)
- **장/절 검색**: 특정 책, 장, 절을 선택하여 정확한 구절 확인
- **키워드 검색**: 성경 전체에서 특정 단어가 포함된 모든 구절 검색 (성구사전 기능)

### 4. 찬양 (Praise)
- 찬송가 번호 또는 제목으로 검색
- 카테고리별(예배, 찬양, 감사 등) 찬양 탐색 및 재생(시뮬레이션)

### 5. 단어 사전 (Word Dictionary)
- 성경 속 용어의 의미를 찾고 관련 구절을 검색

### 6. 원어 성경 (Original Bible)
- 히브리어 및 헬라어 주요 단어 학습 및 검색

### 7. 성경 지도/가이드 (Bible Guide)
- 구약/신약 성경의 각 책별 장수 정보 제공 및 클릭 시 해당 책으로 이동

### 8. 설교 작성 (Sermon Prep) - 핵심 기능
- **5단계 프로세스**:
    1. 주제/성구 입력
    2. AI 추천 제목 선택
    3. 설교 시간 선택
    4. 3대지 개요 자동 생성 (시뮬레이션)
    5. 관련 찬송가 추천
- **최종 편집**: 제목, 본문, 개요, 내용을 통합 편집하고 TXT/JSON 다운로드 및 인쇄 지원

### 9. 내 설교 (My Sermons)
- 작성한 설교를 로컬 저장소에 저장하고 관리 (수정, 삭제, 보기, 인쇄)

### 10. 설교 게시판 (Board)
- 사용자 간 설교 공유 및 추천 기능
- 카테고리별 필터링 지원

## 4. 데이터 구조
- **성경 데이터**: `public/bible/` 경로에 각 권별 JS 파일로 존재
- **메타데이터**: `public/bible-data.js`에 책 목록 및 약어 정의
- **사용자 데이터**: `localStorage`를 이용한 클라이언트 사이드 데이터 저장 (인증, 설교, 게시글)
