
# 성경 읽기 애플리케이션

## 개요

이 애플리케이션은 사용자가 웹 브라우저를 통해 성경을 쉽게 읽을 수 있도록 제작되었습니다. 사용자는 원하는 성경(책)과 장을 선택하여 해당 내용을 확인할 수 있습니다.

## 디자인

- **레이아웃:** 상단에 헤더, 그 아래에 성경 선택 및 장 선택 드롭다운 메뉴, 그리고 선택된 성경 본문이 표시되는 구조입니다.
- **스타일:** 전체적으로 깔끔하고 단순한 디자인을 적용하여 가독성을 높였습니다. `style.css` 파일에 모든 스타일이 정의되어 있습니다.

## 기능

- **성경 선택:** 드롭다운 메뉴를 통해 원하는 성경을 선택할 수 있습니다.
- **장 선택:** 선택된 성경에 따라 해당 성경의 장 목록이 드롭다운 메뉴에 동적으로 생성됩니다.
- **본문 보기:** 성경과 장을 선택하면 해당 본문이 화면에 표시됩니다.

## 프로젝트 구조

- `index.html`: 애플리케이션의 기본 HTML 구조를 정의합니다.
- `style.css`: 애플리케이션의 디자인을 담당하는 CSS 파일입니다.
- `main.js`: 성경 데이터 로드, 드롭다운 메뉴 생성, 본문 표시 등 애플리케이션의 핵심 로직을 담당하는 JavaScript 파일입니다.
- `public/bible/`: 각 성경의 데이터가 JavaScript 파일(`genesis.js`, `exodus.js` 등) 형태로 저장되어 있는 디렉토리입니다.
